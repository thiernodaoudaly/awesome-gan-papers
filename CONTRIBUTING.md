# Contributing to Awesome GAN Papers

First off, thank you for considering contributing!
Your help makes this repository more valuable for the entire AI community.

## How to Contribute

You can contribute in several ways:

* Add a new relevant paper
* Improve descriptions
* Fix broken links
* Improve formatting or organization

## Guidelines for Adding Papers

To maintain quality and consistency, please follow these rules:

### Paper Requirements

* Must be **relevant to GANs or closely related fields** (e.g., diffusion, generative models)
* Prefer **influential or widely cited papers**
* Must include a **link to the paper** (preferably arXiv)
* Include a **short and clear description**

### Format

Please follow this exact format:

```md
- **[Paper Title](LINK)** (Year)  
  > Short description of the contribution
```

### Example

```md
- **[StyleGAN](https://arxiv.org/abs/1812.04948)** (2019)  
  > Introduces a style-based generator architecture for high-quality image synthesis.
```

## Where to Add Your Paper

* Place the paper in the **most relevant section**
* If no section fits, feel free to:

  * Create a new section
  * Or suggest it in your Pull Request

## What Not to Add

* Low-quality or unclear papers
* Irrelevant topics
* Blog posts (unless in Resources section)
* Duplicate entries

## Contribution Process

1. Fork the repository
2. Create a new branch (`feature/add-paper-name`)
3. Make your changes
4. Commit with a clear message
5. Open a Pull Request

## Style Guidelines

* Keep descriptions **short and informative**
* Use consistent Markdown formatting
* Keep the list clean and readable

## Code of Conduct

Be respectful and constructive in all interactions.
We aim to build an inclusive and helpful community.

## Support

If you like this project:

* Give it a star on GitHub
* Share it with others

---
Thanks again for contributing!
